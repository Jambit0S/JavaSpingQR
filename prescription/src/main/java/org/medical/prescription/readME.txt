Данный проект представляет из себя сервер для работы с электронными рецептами
Сервер реализован при помощи фреймворка Spring и немного Lombok

Каждая папка представляет свою часть работы
Где внутри каждой находится по 5 папок, где классы\интерфесы разделены на группы
    Drug - не материальный лечащий препарат(ЛП), общие особености линейки
    Drug_form - реально существующий ЛП. Его особености: форма, кол. и т.д.
    MKB - Международная классификация болезней [XYY.ZZ]
    Patient - классы\интерфейсы связанные с пациентом (в том числе "загулшки" на подпие Medical_Card)
    Prescription - классы\интерфесы связанные с рецептами (в том числе "загулшки" на подпие Doctor)
